DevLog: Hamburger Layout Added

> Log Date: 250503

This update documents the addition of a top-left hamburger menu across all main pages and new HTML navigation within each blog post for seamless reader experience.

I updated the HTML templates across my Astro site to include a responsive hamburger menu in the top left corner. The menu expands into a dropdown navigation box that links to all major sections of Arynwood: the terminal homepage, gallery views, projects, blog, and contact info.

New Hamburger Navigation Layout

This is part of a larger effort to unify the user experience and keep the terminal theme consistent across pages. Each blog post now includes the menu so readers can return to other parts of the site without hitting the back button or getting lost. The HTML template is mobile-first, lightweight, and styled with my lavender terminal aesthetic.

All new DevLogs will include this layout moving forward, and I’m retrofitting older entries to match. This decision was made to enhance cohesion and reduce bounce rates across blog entries.

Next Steps

Now that the hamburger layout is active, my next goal is to refactor some of the content in /blog/index.astro to ensure each DevLog entry clearly shows the excerpt and date. I also want to standardize the way project documentation appears inside each log.

Snippet: Menu Button Script


document.addEventListener('DOMContentLoaded', () => {
  const toggle = document.getElementById('menuToggle');
  const box = document.getElementById('menuBox');

  toggle.addEventListener('click', () => {
    box.classList.toggle('show');
  });

  document.addEventListener('click', (e) => {
    if (!box.contains(e.target) && e.target !== toggle) {
      box.classList.remove('show');
    }
  });
});
