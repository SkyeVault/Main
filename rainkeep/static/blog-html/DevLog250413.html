DevLog 2025-04-14 – Arynwood 3D Gallery Setup

Project Goal

Set up a fully functional 3D gallery using Three.js and Vite, with OrbitControls, lavender floors, charcoal walls, and eventually glowing doors and artwork placement.

Progress So Far

  • Created Vite project in main/rainkeep/public/gallery3d
  • Installed Three.js via npm install three
  • Replaced CDN imports with local ES module imports
  • Set up OrbitControls with working camera drag and zoom
  • Confirmed everything works in dev mode at localhost:5173

Vite Config

We are deploying to arynwood.com/gallery3d, so vite.config.js must include:

export default {
  base: '/gallery3d/',
};

Deployment Issue

Gallery works locally, but shows a white screen on deployment. This is due to path resolution. Current hypothesis: paths are not aligning between Vite base config and Netlify.

What to Do Next

  1. Ensure vite.config.js is in gallery3d/, not higher up
  2. Run npm run build
  3. Confirm dist/index.html has script paths starting with /gallery3d/
  4. Deploy dist/ to Netlify with publish directory set to dist
  5. If testing locally: change vite.config.js base to './' and run serve dist
